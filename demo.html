üéì How It Works

1. Parse Configuration

Extracts parameters from the data attribute using regex pattern matching. Supports the format: key:`value`;key:`value`;

2. Normalize URL

Intelligently detects protocol (http://, https://, data:, blob:) and adds https:// if missing from domain-like URLs.

3. Validate Input

Checks that the required link parameter exists. Shows a helpful error modal if validation fails.

4. Create Blob

Generates a self-contained HTML blob with an embedded iframe, all metadata (title, favicon), and optimized CSS.

5. Replace History

Uses location.replace() to seamlessly swap the current page with the blob URL, maintaining clean browser history.

6. Spoof Metadata

Updates document title, favicon, and uses History API to hide the iframe implementation from user awareness.
