✅ s.js Script Tests

Below are different test cases showing correct usage of the s.js script.

Note: Click a button to test - it will redirect the page to embedded content.

Test 1: Simple Domain (Auto HTTPS)

Format: link`google.com`

This uses the simplest format - just the domain name, automatically converts to https://

<script src='https://kbsigmaboy67.github.io/-/s.js' 
  data='link`google.com`' />

Test 2: Full HTTPS URL with Custom Title

Format: link`https://example.com`;tab-name`Custom Title`

Full URL with custom browser tab title

<script src='https://kbsigmaboy67.github.io/-/s.js' 
  data='link`https://example.com`;tab-name`My Portal`' />

Test 3: Backtick + Colon Format

Format: link:`https://example.com`

Alternative format with colon between key and value

<script src='https://kbsigmaboy67.github.io/-/s.js' 
  data='link:`https://example.com`;tab-name:`Example`' />

Test 4: Simple Colon + Semicolon Format

Format: link:value;tab-name:value;

Plain key:value format with semicolons

<script src='https://kbsigmaboy67.github.io/-/s.js' 
  data='link:https://example.com;tab-name:Example Site;' />

Test 5: Full Featured Example

Format: All parameters with custom favicon

Complete example with title and favicon

<script src='https://kbsigmaboy67.github.io/-/s.js' 
  data='link`https://github.com`;tab-name`GitHub`;tab-image`https://github.com/favicon.ico`' />

Test 6: Query Parameters in URL

Format: link`example.com?param=value&other=123`

URLs with query parameters work correctly

<script src='https://kbsigmaboy67.github.io/-/s.js' 
  data='link`google.com/search?q=test`' />

⚠️ Important: These tests will redirect your page. To see the s.js script in action without leaving this page, copy one of the code examples above and paste it into your own HTML file.
